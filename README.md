# FastMes

Мессенджер с чатом в реальном времени, друзьями и звонками (голос/видео).

## Структура проекта

```
FastMesServer/
  server.js       — сервер (Express + Socket.io)
  package.json     — зависимости
  data.json        — база данных (создаётся сама при первом запуске)
  public/
    index.html
    style.css
    client.js
```

## Как запустить у себя локально (проверить перед деплоем)

1. Установи Node.js (nodejs.org), если ещё нет
2. В папке проекта:
   ```
   npm install
   npm start
   ```
3. Открой в браузере: http://localhost:3000

## Как выложить на Render.com (бесплатно)

1. Залей папку `FastMesServer` в свой репозиторий на **GitHub** (создай новый репозиторий → загрузи файлы, либо через git push)
2. Зайди на **render.com** → войди/зарегистрируйся
3. Нажми **"New +" → "Web Service"**
4. Подключи свой GitHub-репозиторий с проектом
5. Настройки:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Instance Type:** Free
6. Нажми **Create Web Service** — через пару минут получишь ссылку вида `https://твой-проект.onrender.com`

## Чтобы сервер не "засыпал" (как советовал дядя)

Бесплатный тариф Render усыпляет сервер после ~15 минут без запросов.
Чтобы он не засыпал:

1. Зайди на **UptimeRobot.com** (бесплатно) или **cron-job.org**
2. Создай монитор/задачу, которая раз в 5-10 минут открывает твою ссылку (`https://твой-проект.onrender.com`)
3. Готово — сервер всегда будет "бодрым"

## Важно про данные

Все аккаунты, друзья и сообщения хранятся в файле `data.json` прямо на сервере.
При обычной работе (сон/пробуждение) данные сохраняются.
Но если ты **вручную пересоздашь сервис** на Render (не просто redeploy, а удаление и создание заново) — файл данных исчезнет. Для учебного проекта на двоих это не проблема, но помни об этом.

## Звонки

Голосовые и видеозвонки работают через WebRTC напрямую между браузерами (сервер только помогает "договориться" о соединении). Может не работать в редких случаях со строгими мобильными сетями/корпоративными Wi-Fi — для таких случаев в будущем можно подключить платный TURN-сервер.
